# 06_Tugas-Socket-Programming
Repository ini berisi program Tugas Socket Programming mata kuliah Jaringan Komputer II2120 tahun 2024

## Requirements
Python 3.10
Socket library
Threading library
tkinter 
csv

## How to run
### Server
````
cd <file-path>
````

then run
````
python3 serverGUI.py
````

### Client
````
cd <file-path>
````

then run
````
python3 clientGUI.py
````
